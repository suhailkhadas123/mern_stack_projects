# Laundry Service Website (Responsive UI)

This project is a **responsive laundry service landing page** built using **HTML and CSS**. It includes a modern navbar and hero section that adapt across desktop, tablet, and mobile devices.

---


###  Responsive Navbar
- Uses **Flexbox** for layout
- Desktop: Logo + Navigation Links + Username
- Tablet: Reduced font sizes
- Mobile:
  - Navigation links are hidden
  - Only logo and username are displayed

---



## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
